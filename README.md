# Python 爱盘爬虫工具
一款基于Python的吾爱破解论坛爱盘下载工具
改自MikoSec原版

本工具支持：
1. 自动重试下载
2. 连接、下载错误处理
3. 多层文件夹下载

# 鸣谢

# 安装、使用方法
本工具使用Python3，requests库，BeautifulSoup4库以及lxml库
- [Python3](https://www.python.org/downloads/)

所有库均可通过Python内置的pip工具下载
- pip install *package name*

建议使用前先修改User Agent
